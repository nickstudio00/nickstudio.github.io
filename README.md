# 환영합니다!
파이썬 가이드입니다.



# 1. 파이썬 설치
1. https://www.python.org/downloads/ 으로 들어갑니다.
2. 자신의 운영체제에 맞는 파일을 다운받습니다.
3. 다운로드가 완료된 후 다운로드한 파일을 엽니다.
4. Install Now를 누르면 설치가 진행됩니다.

# 2. 파이썬 실행해보기

Window 운영체제<br>
1. 윈도우 키를 눌러 윈도우 창을 엽니다.
2. 검색창에 cmd(커맨드라인)을 검색한 후 실행합니다.
3. (명령 프롬포트 실행)
4. python을 친 후 enter 키를 누릅니다.
5. 그러면 파이썬 준비가 끝났습니다. 예시로 print("hello world")를 쳐 봅시다.
6. 복사하는것보다 직접 쳐보는것이 도움이 될 것입니다

Mac os 운영체제<br>
1. 터미널을 실행시킵니다.
2. python을 친 후 enter 키를 누릅니다.
3. 그러면 파이썬 준비가 끝났습니다. 예시로 print("hello world")를 쳐 봅시다.
4. 복사하는것보다 직접 쳐보는것이 도움이 될 것입니다

# 3. 인터프린터
정의<br>

인터프린터는 프로그래밍 언어의 소스코드를 바로 실행하는 컴퓨터 프로그램 또는 환경을 말합니다
파이썬의 인터프린터는 두가지 종류가 있습니다.
<br>
<br>

대화형 인터프린터<br>

대화형 인터프린터에서는 입력에 대한 출력 값이 바로 나옵니다.
입력과 출력이 번갈아 이어지는 것이 마치 대화하는 것과 같다고 하여 ‘대화형’ 인터프리터라고 합니다.
파이썬 IDLE를 설치라여 사용할수 있습니다.
<br>
<br>

코드 편집기<br>

일반적으로 개발 환경에서 사용됩니다.
프로그램의 전체 소스 코드를 입력하고 입력된 소스 코드 전체가 실행됩니다.
파이참이나 VS코드같은 프로그램으로 사용할수 있습니다.
