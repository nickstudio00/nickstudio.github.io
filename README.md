# 1. 파이썬 설치
1. https://www.python.org/downloads/ 으로 들어갑니다.
2. 자신의 운영체제에 맞는 파일을 다운받습니다.
3. 다운로드가 완료된 후 다운로드한 파일을 엽니다.
4. Install Now를 누르면 설치가 진행됩니다.

# 2. 파이썬 실행해보기

Window 운영체제
1. 윈도우 키를 눌러 윈도우 창을 엽니다.
2. 검색창에 cmd(커맨드라인)을 검색한 후 실행합니다.
3. (명령 프롬포트 실행)
4. python을 친 후 enter 키를 누릅니다.
5. 그러면 파이썬 준비가 끝났습니다. 예시로 print("hello world")를 쳐 봅시다.
6. 복사하는것보다 직접 쳐보는것이 도움이 될 것입니다

Mac os 운영체제
1. 터미널을 실행시킵니다.
2. python을 친 후 enter 키를 누릅니다.
3. 그러면 파이썬 준비가 끝났습니다. 예시로 print("hello world")를 쳐 봅시다.
4. 복사하는것보다 직접 쳐보는것이 도움이 될 것입니다
